# fnwebsite

Static Astro marketing site for Food Ninja.

## Sections

- `/{locale}/` - Overview (single page)
- `/{locale}/features/` - Features hub
- `/{locale}/features/{slug}/` - Feature detail pages
- `/{locale}/pricing/` - Pricing section
- `/{locale}/about/` - About section root
- `/{locale}/about/tos/` - Terms template
- `/{locale}/about/pp/` - Privacy template
- `/{locale}/about/contact-us/` - Contact section

## Legal Text Files

- `public/legal/tos-en.txt`
- `public/legal/tos-zh-TW.txt`
- `public/legal/pp-en.txt`
- `public/legal/pp-zh-TW.txt`

Terms/Privacy pages include download buttons for these files.

Supported locales: `en`, `zh-TW`.

## Dev commands

- `npm run dev`
- `npm run build`
- `npm run preview`
User is not intended to view repo.
